# Shell中简单for循环的几种形式

1.变量 ${sample}
```
for sample in name1 name2 name3
do
	${sample}.input > result_${sample}.output
done
```

## Tricks

- 连续多行变为一行
> paste - - - (连续三行变为一行)

- awk 传入参数
> awk -v value=${sample}
